# HR Database SQL Script

Dieses Repository enthält ein SQL-Skript zur Erstellung und Verwaltung der HR-Datenbank mit einer PERSON-Tabelle.

## Inhalt der Tabelle PERSON

ID, Vorname, Nachname, Geschlecht, Geburtsdatum

Einstellungsdatum, Austrittsdatum

Abteilung, Position, Manager

Telefonnummer, Gehalt

## Beispiele im Skript

Mitarbeiter anzeigen (Name, Gehalt, Geburtsdatum)

Neue Mitarbeiter einfügen

Gehalt aktualisieren und erhöhen

Mitarbeiter löschen

Filter nach Abteilung, Gehalt oder Geburtsdatum

Sortieren und Top 5 Gehälter anzeigen

Anzahl der Abteilungen zählen
